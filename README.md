# Turkcell One — Paket Sayfaları (klon)

Gerçek Turkcell One paket listeleme ve paket detay sayfalarının bire bir, çalışan klonu.

- index.html — paket listeleme / karosel (zenginleştirilmiş kartlar: platform logoları + dahil liste + öne çıkan)
- detay.html — paket detay (platformlar + öne çıkanlar + 3 adımda abone + SSS)
- style.css / app.js — ortak stil ve etkileşimler
- assets/ — gerçek Turkcell + platform + partner logoları

Statik site; Vercel: Framework=Other, output=kök.
